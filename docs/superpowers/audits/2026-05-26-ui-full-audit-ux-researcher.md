## HIGH 严重问题

### 1. 无 PIN 找回机制
→ 添加 "如果忘记 PIN，请重新注册" 提示，或安全问题找回

### 2. 个人信息硬性门槛
所有工具页 `missingBirthInfo` 时显示死胡同页面
→ 展示示例数据，加 "完善出生信息以查看个人结果" 横幅

### 3. 专业术语无内联词汇表
十神、藏干、纳音、六亲、六神 等无 hover tooltip
→ 实现 TermTooltip 组件，一句话白话解释每个术语

### 4. 八字解读被埋在页面最底部
→ 移到页面顶部，重命名为 "命理速览"

### 5. 分数解读缺乏上下文
六爻评分、流年评分无图例解释
→ 加分段标签：80-100=大吉，60-79=吉，40-59=中平，20-39=凶，0-19=大凶

---

## MED 严重问题

### 无演示/示例模式
→ 登录页加 "演示模式" 加载示例数据

### 首页工具描述过于抽象
→ 改为面向用户利益的描述

### 移动端导航栏消失（hidden md:flex）
→ 添加底部导航栏

### 已保存反馈无加载状态
→ 添加 "保存中..." 三态指示器

### 起卦进度无取消确认
→ 已掷5次后按"重新起卦"加确认对话框

---

## Top 5 改进建议

1. **内联术语提示** — 所有专业术语加 hover tooltip
2. **演示模式 + 新手引导** — 降低入门门槛
3. **八字解读移到顶部** — "概览在前，详情按需"
4. **移动端底部导航栏** — 减少导航摩擦
5. **PIN 找回 + 登录体验优化** — 消除最关键的流程死胡同
