## 第 3 批：核心新功能 ✅ 已完成

### 3.1 八字合婚

**难度：** ✦✦✦ (中)

**功能描述：** 输入双方出生信息，从八个维度分析婚配吉凶。

**维度：**
1. **生肖配对** — 三合 / 六合 / 相冲 / 相刑 / 相害
2. **年柱关系** — 天干相生克、地支相合冲
3. **日主五行** — 日主之间的生克关系
4. **十神匹配** — 男命妻星 vs 女命夫星
5. **大运同步** — 双方大运走势是否同步
6. **子女信息** — 子女星对比
7. **纳音配对** — 年命纳音五行关系
8. **冲刑害合** — 四柱逐柱对比

**技术方案：**
1. 新页面 `pages/tools/marriage.vue`
2. 复用 `useBaZi.ts` 引擎，双方各自排盘后对比
3. 结果页：综合评分环 + 各维度详细分析 + 总评规则模板

### 3.2 姓名测试·五格剖象

**难度：** ✦✦✦ (中高)

**功能描述：** 输入姓名（姓 + 名），计算天格/人格/地格/外格/总格五格数理，结合五行分析姓名吉凶。

**技术方案：**

1. 数据源：max32002/chinese_dictionary（开源 JSON，81k 汉字含 `strokes_total` 字段）。执行时下载 Dictionary_lite.json，提取 GB2312 常用字笔画数，生成 `constants/stroke-data.ts`（Record<字符, 笔画数>）
2. 新页面 `pages/tools/naming.vue`
3. 计算逻辑：纯前端 TypeScript
4. 展示：五格数理表 + 五行相生 + 三才配置 + 总评

**关键数据：** 康熙笔划字典（来自 max32002/chinese_dictionary）+ 81 数理吉凶表（1-81 大吉/半吉/凶 + 释义，已搜索确认完整数据）

## 第 5 批：意境 + 复杂功能（估算：2-3 周）

### 5.1 测字占卜

**难度：** ✦✦ (低)

**功能描述：** 用户写一个字，按笔画数/结构/五行拆解，给出吉凶断语。

**技术方案：**
1. 输入：单字输入框，支持手写或键盘
2. 拆解：按康熙笔划、五行属性、字形结构（独体/左右/上下/包围）
3. 断语：规则模板匹配
4. 新页面 `pages/tools/divination-char.vue`
5. **美学：** 毛笔书写效果，纸笺展示，与书房意境绝配

### 5.2 每月运势

**难度：** ✦✦ (低)

**功能描述：** 在生肖页增加每月运势分解，以月为维度展示各月吉凶。

**技术方案：**
1. 复用流年计算逻辑
2. 按月展示：旺月/平月/弱月 + 一句话提示
3. 可折叠月度列表

### 5.3 用户系统增强

| 改进项 | 说明 |
|--------|------|
| 出生地字段 | 影响真太阳时，八字/紫微/星盘都需要 |
| 头像 | 简笔头像或默认生成 |
| 多档案管理 | 管理家人/朋友档案，切换查看 |
| 操作历史 | 首页最近使用记录（曾因设计问题移除，可重新设计） |

### 5.4 导出 / 分享

**难度：** ✦✦ (低)

**功能描述：** 各工具结果可保存为图片或 PDF，方便分享。

**技术方案：**
1. 使用 `html2canvas` 或 `html-to-image`
2. 各工具结果区增加"存为图片"按钮
3. 结果卡片的墨韵风格导出模板

### 5.5 择吉日

**难度：** ✦✦✦✦ (高)

**功能描述：** 根据用户选择的事项（嫁娶/开业/搬家/出行等），结合黄历建除、二十八宿、黄黑道、时辰吉凶选出吉日。

**技术方案：**

1. 数据源：复用项目中已安装的 `lunar-javascript@1.7.7`，其 `Lunar` 类直接提供 `getDayTwelveStar()`（建除十二值星）、`getXiu()`（二十八宿）、`getDayYi()`/`getDayJi()`（宜忌）等方法，无需外部依赖
2. 日历组件：展示当月每日吉凶标记
3. 推荐日期列表：按事项类型筛选匹配的吉日
