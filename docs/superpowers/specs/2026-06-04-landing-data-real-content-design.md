# Landing Page 真实数据替换设计

## 概述

将 landing page（`pages/index.vue`）中 3 个使用硬编码假数据的模块替换为基于真实项目引擎的动态数据或准确的内容描述，消除「展示信息与实际项目出入很大」的问题。

## 涉及模块

### 1. 命盘预览 → 实盘展示

**现状**：硬编码甲辰·丙寅·戊午·壬子四柱，五行百分比随意，解读文案写死。

**改造方案**：

- 在 `<script setup>` 中定义固定示例生日（`1990-05-15 12:00`），调用 `useBaZi` 引擎的 `calculateBaZi()` 生成真实四柱、五行分布、十神、纳音、藏干，调用 `useShenSha` 的 `calculateShenSha()` 添加神煞信息。
- 渲染使用现有的 `grid-cinnabar` 表格和 `card-warm` 卡片，保持墨韵视觉不变。
- 底部解读文案采用规则模板拼接（复用 `useLiuNian` 的文案风格）。
- 五行柱百分比来自引擎实际统计（各五行在四柱中出现的加权次数）。
- 表格上方显示"示例：1990年5月15日 午时"，左下角加标注"\* 此为示例命盘，登录后可排自己的盘"。

### 2. 今日运势 → 今日玄机

**现状**：日期写死（五月三十·星期日），运势文案虚构。

**改造方案**：

- 标题改为"今 日 玄 机"。
- 使用真实日期：公历日期 → `lunar-javascript` 农历转换 → 当日节气（`useSolarTerms.ts` 的 `getSolarTerm()`）→ 当日年柱月柱。
- 展示形式：卡片内左文，竖排三段式信息，更具仪式感。
- 底部增加 CTA："登录查看个人运势"。

### 3. 命簿卡片 → 四象推演

**现状**：两个假卡片展示虚构的性格分析和生肖配对。

**改造方案**：

- 标题改为"四 象 推 演"或"术 法 维 度"。
- 四个卡片 2×2 网格：
  - **八字**：四柱推命 · 十神格局 · 大运流年
  - **六爻**：起卦断事 · 爻辞解读 · 变卦推断
  - **生肖**：生肖性格 · 五行属性 · 相性配对
  - **星座**：星盘轨迹 · 今日宜忌 · 缘分深浅
- 每个卡片带对应卦象角标（☰☵☷☲），与上方工具卡片呼应。
- 采用 `card-warm` 样式，保持墨韵设计系统一致。

## 不涉及改动

- Hero（品牌展示区）：不动，保持"玄·道知天命" + 咒语式 tagline。
- 术数工具网格：不动，工具卡片内容准确。
- 已登录态的推演工具网格：不动。
- `layouts/default.vue`、`pages/login.vue`：不动。
- `tailwind.config.ts`、`assets/css/main.css`：不动（现有样式已覆盖）。

## 成功标准

1. 命盘预览展示的八字、五行、十神、纳音全部来自引擎真实计算
2. 今日玄机显示的日期、节气、日柱与当天一致
3. 功能亮点卡片内容准确描述各工具的实际能力
4. 视觉风格与现有墨韵设计系统一致，无视觉断裂
5. 构建/类型检查/测试全部通过
